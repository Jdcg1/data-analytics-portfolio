# Data Analytics Portfolio – Daniel González

Este repositorio contiene proyectos prácticos de análisis de datos enfocados en:
- Limpieza y preparación de datos
- Análisis exploratorio
- Toma de decisiones basadas en datos
- Uso de Excel

## Herramientas utilizadas
- Microsoft Excel
- Funciones y fórmulas
- Tablas y filtros
- Análisis descriptivo

## Proyectos
1. Análisis de ventas de un pequeño negocio
2. Limpieza y preparación de datos de clientes
3. Análisis para la toma de decisiones comerciales
