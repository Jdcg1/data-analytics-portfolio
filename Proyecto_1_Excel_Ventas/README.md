# Proyecto 1 – Análisis de Ventas Mensuales

## Objetivo
Analizar las ventas mensuales de un pequeño negocio para identificar productos más rentables y tendencias de venta.

## Herramientas
- Microsoft Excel

## Enfoque
- Análisis de datos de ventas
- Uso de fórmulas básicas
- Organización y resumen de información

## Resultados
- Identificación de productos con mayor ingreso
- Detección de la categoría más rentable
- Recomendaciones para optimizar ventas
